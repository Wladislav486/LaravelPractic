Для создания админки будем использовать готовый стилевой шаблон
https://adminlte.io/themes/v3/

1 Разместили во view html пустой страницы blank
2 Сверху пошли по подключаемым файлам и перенесли в наш проект в
resources/assets/admin
3 копируем front-файлы в public
с помощью внесения логики в испольняемый файл laravel mix - webpack.mix.js

1 set NODE_OPTIONS=--openssl-legacy-provider в терминале виндовс, при возникновении ошибки
