Сущности
1 категории постов
2 посты
3 теги

Связи
1  один(категория) ко многим(посты)
- ссылка на категорию в постах
2 многие(посты) ко многим(тегов)
- табличка связей

Создание бд
1 Создаём миграции на создание таблиц и модели
php artisan make:model Category -m
php artisan make:model Tag -m
php artisan make:model Post -m
2 Создаём таблицу для связи многие ко многим(посты и теги)
- post_tag по конвенции
php artisan make:migration create_post_tag_table
3 Добавляем создание необходимых полей в миграции
4 Создание св-в ссылок на связные сущности для моделей
5 Заполнение поля slug(транслитирация+уникальное)
- Транслетирация - helper laravel Str::slug
- уникальное - пакет laravel sluggable
